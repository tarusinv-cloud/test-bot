# Server-Side Cursor

Серверное приложение для управления курсором. Состояние курсора (позиция, видимость, стиль) полностью хранится и управляется на сервере. Клиенты подключаются через WebSocket для получения обновлений в реальном времени и отправки команд.

## Возможности

- **Серверное состояние** — курсор живёт на сервере; все клиенты видят одно и то же.
- **WebSocket** — мгновенная синхронизация между всеми подключёнными клиентами.
- **REST API** — программный доступ к управлению курсором.
- **Веб-интерфейс** — визуализация и управление курсором прямо в браузере.
- **Настройка стиля** — цвет, размер, метка курсора.

## Быстрый старт

```bash
pip install -r requirements.txt
python main.py
```

Сервер запустится на `http://0.0.0.0:8000`. Откройте в браузере для доступа к интерфейсу.

## REST API

| Метод | Путь | Описание |
|-------|------|----------|
| `GET`  | `/api/cursor` | Получить текущее состояние курсора |
| `POST` | `/api/cursor/move?x=100&y=200` | Переместить курсор |
| `POST` | `/api/cursor/toggle` | Переключить видимость |
| `POST` | `/api/cursor/style?color=%23ff0000&size=24&label=admin` | Изменить стиль |

## WebSocket

Подключение: `ws://host:8000/ws`

### Команды (клиент → сервер)

```json
{"action": "move", "x": 100, "y": 200}
{"action": "toggle"}
{"action": "style", "color": "#00ff00", "size": 24, "label": "user1"}
```

### События (сервер → клиент)

```json
{"type": "cursor_update", "state": {"x": 100, "y": 200, "visible": true, "color": "#e74c3c", "size": 18, "label": "", "last_updated": 1700000000.0}}
```
